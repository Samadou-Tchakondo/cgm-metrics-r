# Computing Continuous Glucose Monitoring (CGM) Metrics in R

This repository presents a methodological and reproducible framework for computing Continuous Glucose Monitoring (CGM) metrics in R.

## Included Metrics

- Mean Absolute Relative Difference (MARD)
- Continuous Overall Net Glycemic Action (CONGA)
- Mean Amplitude of Glycemic Excursions (MAGE)
- Mean of Daily Differences (MODD)

## Main Features

- CGM data structure requirements
- Reproducible R workflows
- Methodological explanations
- Practical implementation examples
- Visualization workflows using the `iglu` package

## Technologies Used

- R
- R Markdown
- iglu
- dplyr
